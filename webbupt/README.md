# 北京邮电大学七十周年校庆网站

## 项目简介

这是一个为北京邮电大学七十周年校庆设计的完整静态网站，展示了北邮的历史、成就、校园风采等内容。

## 项目结构

`
.
 index.html              # 首页
 about.html              # 关于我们 - 北邮历史
 services.html           # 北邮成就
 gallery.html            # 校园风采
 contact.html            # 联系我们
 css/
    style.css           # 样式表
 images/                 # 图片文件夹
    gallery_item1-4.svg # 校园图片占位符
    create_placeholders.html
 media/                  # 媒体文件夹
    README.md           # 媒体文件说明
 README.md               # 本文件
`

## 功能特性

### 首页 (index.html)
- 北邮七十周年校庆主题
- 英雄区展示校庆主题
- 四个特性卡片展示北邮成就

### 关于我们 (about.html)
- 北邮历史介绍
- 发展历程（1955年至今）
- 学校成就展示

### 北邮成就 (services.html)
- 四个成就领域卡片
- 十个成就荣誉表格行
- 详细的成就介绍

### 校园风采 (gallery.html)
- 四张校园图片
- 校歌音频
- 校庆祝福视频

### 联系我们 (contact.html)
- 作者个人信息
- 留言反馈表单

## 设计特点

- **北邮蓝色主题** - 统一的品牌色彩
- **现代化设计** - 优雅的动画和过渡效果
- **完全响应式** - 支持所有设备
- **语义正确** - HTML5标准
- **易于维护** - 清晰的代码结构

## 快速开始

1. 在浏览器中打开 index.html
2. 浏览各个页面
3. 根据需要修改内容

## 自定义指南

### 修改内容
编辑相应的HTML文件即可修改文本内容。

### 修改样式
编辑 css/style.css 文件修改样式。

### 添加媒体文件
- 校园照片  images/ 文件夹
- 校歌音频  media/ 文件夹
- 校庆视频  media/ 文件夹

### 更新作者信息
编辑 contact.html 中的作者个人信息部分。

## 技术规格

- HTML5 标准
- CSS3 特性
- 无JavaScript依赖
- 完全静态网站
- 浏览器兼容：Chrome, Firefox, Safari, Edge

## 部署

将所有文件上传到服务器即可，无需任何配置。
