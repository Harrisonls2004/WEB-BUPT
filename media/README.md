# 媒体文件说明

此文件夹用于存放网站的音频和视频文件。

## 所需文件

### 1. 校歌音频 (school_song.mp3)
- 文件名：`school_song.mp3`
- 格式：MP3
- 用途：在画廊页面播放北邮校歌

### 2. 校庆祝福视频 (anniversary_video.mp4)
- 文件名：`anniversary_video.mp4`
- 格式：MP4
- 用途：在画廊页面播放校庆祝福视频

## 如何添加文件

1. 准备您的音频和视频文件
2. 将文件放入此文件夹
3. 确保文件名与上述要求一致
4. 刷新网页即可看到媒体内容

## 文件格式建议

### 音频
- 格式：MP3
- 比特率：128-192 kbps
- 采样率：44.1 kHz

### 视频
- 格式：MP4 (H.264 编码)
- 分辨率：1280x720 或更高
- 比特率：2-5 Mbps
- 帧率：30 fps

## 免费资源

如果您需要示例文件，可以从以下来源获取：
- 音频：Pixabay, Freepik, YouTube Audio Library
- 视频：Pixabay, Pexels, Unsplash

## 注意事项

- 确保您有权使用这些媒体文件
- 文件大小不要过大，以保证网页加载速度
- 建议在上传前进行测试
