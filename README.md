## 📖 网站简介
这是一个为北京邮电大学七十周年校庆精心设计的现代化静态网站，全面展示了北邮七十年来的辉煌历程、卓越成就和美丽校园风采。网站采用北邮蓝色主题，融合现代化设计理念，为校友、师生和社会各界提供一个了解北邮、感受北邮精神的窗口。

## 🛠️ 自定义指南

### 修改文本内容

**首页内容**
- 编辑 `index.html`
- 修改英雄区的诗句和口号
- 更新数据统计卡片的数字

**历史内容**
- 编辑 `about.html`
- 修改发展历程时间线
- 更新学校成就信息

**成就内容**
- 编辑 `services.html`
- 添加或修改成就卡片
- 更新荣誉表格

### 修改样式

**颜色主题**
```css
/* 在 css/style.css 顶部修改 CSS 变量 */
:root {
    --primary-color: #0066cc;    /* 主色调 */
    --accent-color: #3b82f6;     /* 强调色 */
    --sidebar-bg: #2d3748;       /* 侧边栏背景 */
}
```

**字体设置**
```css
:root {
    --font-heading: 'Microsoft YaHei', sans-serif;
    --font-body: 'Microsoft YaHei', sans-serif;
}
```

**动画速度**
```css
/* 搜索 transition 属性并修改时间 */
transition: all 0.3s ease;  /* 改为 0.5s 会更慢 */
```

### 添加媒体文件

**添加校园图片**
1. 将图片放入 `images/` 文件夹
2. 在 `gallery.html` 中添加：
   ```html
   <figure>
       <img src="images/你的图片.jpg" alt="描述">
       <figcaption class="gallery-caption">图片标题</figcaption>
   </figure>
   ```

**更换校歌音频**
1. 将新的音频文件放入 `media/` 文件夹
2. 在 `gallery.html` 中修改音频源路径

**更换祝福视频**
1. 将新的视频文件放入 `media/` 文件夹
2. 在 `gallery.html` 中修改视频源路径
